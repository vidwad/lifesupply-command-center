## Recommended Use

1. Create a new repository named `lifesupply-command-center`.
2. Place these files at the repository root.
3. Open the repository in VS Code.
4. Start Claude Code in the repo.
5. Paste the prompt from `00_START_HERE_FOR_CLAUDE_CODE.md`.
6. Ask Claude Code to summarize first before writing code.
7. Approve Phase 0 and Phase 1 before any broader implementation.
